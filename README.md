☕ Cafeteria - Landing Page
Esta é uma Landing Page moderna e responsiva para uma cafeteria, desenvolvida como parte do meu portfólio de Desenvolvimento Front-end. O foco do projeto foi aplicar conceitos avançados de Flexbox e Media Queries para garantir uma experiência fluida em dispositivos móveis e desktops.

🚀 Tecnologias e Ferramentas
HTML5: Estruturação semântica (header, main, footer).
CSS3: Estilização personalizada, uso de variáveis e layouts flexíveis.
Responsividade: Breakpoints configurados para telas de até 768px (Mobile-first mindset).
VS Code: Ambiente de desenvolvimento.
Vercel: Deploy e hospedagem contínua.

🛠️ Funcionalidades Implementadas
Menu de Navegação: Links rápidos para as seções do site.
Seção Hero: Destaque para o produto principal com botões de chamada para ação (CTA).
Rodapé Customizado: Organização em colunas com informações de contato e redes sociais, que se ajustam verticalmente em telas menores.
Interatividade: Efeitos de hover nos botões e links.
